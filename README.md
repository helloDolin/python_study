### Linux基本命令

ls:list

ls -l:long list

ls -a:all list 显示所有文件，包括隐藏文件

cd:change directory

cd ~: 进入当前用户home目录

cd ..: 进入上一级目录

cd -: 进入上次所在目录

pwd:print working directory 打印当前目录

cp:copy

mv:剪切/改名

date:显示系统当前时间

mkdir -p 创建多级目录 eg：mkdir -p 123/456/789

touch:创建文件

clear：清屏

history：历史命令
